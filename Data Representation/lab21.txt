Lab 21 reads a value for N from standard input, and N is an uninitialized doubleword. A loop accepts N doublewords from standard input, and each value is pushed onto each of the values to the system stack. Once all of the values have been input, a second loop displays the original input values in reverse order. The original values from the system stack are placed one at a time in the eax register. The values of the register are then dumped afterward.
