Lab 9 is a Turing machine for the following:

(x , x=ans) | x elem {a,b}*, where ans={1 if x contains an odd # of substring 'aa' occurrences, else 0}).

Note: 'aa' is considered to have two occurrences of 'aa'

Other considerations:
1. As each input line is read in, the main function creates the initial tape contents with 10 blanks followed by the original input line followed by more blanks.
2. The halt state is 0, and the start state is 1.
